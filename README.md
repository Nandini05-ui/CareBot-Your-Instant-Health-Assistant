# CareBot – Your Instant Health Assistant

## 🌐 Project Overview
CareBot is a web-based smart health chatbot designed to provide instant guidance for common health symptoms through an interactive chatbot interface.

---

## ❗ Problem Statement
Many people lack immediate guidance for common health issues and may not know what to do in the early stages of symptoms.

---

## 💡 Solution
CareBot provides quick, symptom-based advice using keyword-triggered chatbot responses integrated into a website platform.

---

## 🏗️ System Architecture
CareBot consists of three main components:
- User Interface (Wix Website)
- Chatbot Engine (Landbot Platform)
- Keyword-Based Response System

---

## 🚀 Key Features
- 24/7 Availability  
- Instant Responses  
- Secure & Private Interaction  
- Simple and User-Friendly Interface  

---

## 🩺 Supported Symptoms
- Headache  
- Fever  
- Stomachache  
- Cough  
- Muscle Pain  
- Sore Throat  
- Allergies  
- Cold  
- Dizziness  
- Nausea  
- Fatigue  

---

## ⚙️ How It Works
User Input → Keyword Detection → Symptom Identification → Response Generation → Medical Disclaimer

---

## 🛠️ Technology Used
- Landbot (Chatbot Platform)  
- Wix (Website Integration)  
- Keyword-Based Trigger Logic  

---

## ⚠️ Disclaimer
CareBot provides general health guidance only and is not a substitute for professional medical consultation.
